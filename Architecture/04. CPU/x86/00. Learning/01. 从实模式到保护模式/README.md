《x86 汇编语言 从实模式到保护模式》读书笔记

本书采用开源的 NASM 汇编语言编译器和 VirtualBox 虚拟机软件, 以 Intel 处理器为基础, 详细讲解了 Intel 处理器的指令系统和工作模式, 以大量的代码演示了 16／32／64 位软件的开发方法. 集中介绍处理器的 16 位实模式和 32 位保护模式, 以及基本的指令系统.
