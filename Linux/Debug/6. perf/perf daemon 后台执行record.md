# 参数说明

```
 Usage: perf daemon start [<options>]
    or: perf daemon [<options>]

    -v, --verbose         be more verbose
    -x, --field-separator[=<field separator>]
                          print counts with custom separator
        --base <directory>
                          base directory
        --config <config file>
                          config file path
```
