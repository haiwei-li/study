# 1. 用途

用于读取 perf record 保存的裸 trace 数据.

1. 查看 perf 的数据文件( perf.data )
2. 执行基于 python/perl 的扩展功能 ( 需要 python 环境 )

# 2. 使用方法

```
perf script -h
```

```
perf report [-i <file> | --input=file]
```
