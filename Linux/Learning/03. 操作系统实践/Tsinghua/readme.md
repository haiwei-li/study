
清华大学 操作系统 ucore

一个国内的类似 xv6 的 lab, 那么这里推荐清华大学 ucore OS,  它其实也是起源于 MIT CSAIL PDOS 课题组开发的 xv6 & jos、哈佛大学开发的 OS161 教学操作系统、以及 Linux-2.4 内核.

然后清华的老师在这个基础上, 增加、调整了一些代码, 为清华大学 OS 课程设计的一个 lab: ucore

另外, 这个课程的很多资料、文档都是中文的, 还有清华老师的配套视频课程.
